Flash card using HTML

This Project revolves around a series of questions randomly generated by the code, if we got the answer to the questions wrong our score will be deducted and shuffle the question and if we got the question correct we would gain 1 score and it will shuffle the question again. If we get to a point of -5 the program will display a game over screen and we have the option to restart or quit the program, if we get 10 points the program will instead show a winning screen with wither options to quit the game or to restart
